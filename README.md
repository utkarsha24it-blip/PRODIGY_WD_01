# PRODIGY_WD_01
Responsive Landing Page
This HTML file is a **complete front-end structure of a modern artisan bakery website** called **“La Miètte”**. It is well-organized into multiple sections and designed for a visually rich, user-friendly experience.

Here’s a clear explanation of each part:

---

## 🔷 1. Document Setup (Head Section)

```html
<head>
```

* Defines **character encoding (UTF-8)** and **responsive design (viewport)**.
* Sets the **title** of the website: *La Miètte — Artisan Bakery*.
* Imports **Google Fonts** (`Cormorant Garamond` and `Jost`) for elegant typography.
* Links an external **CSS file (`style.css`)** for styling.

---

## 🔷 2. Navigation Bar (Navbar)

```html
<nav id="navbar">
```

* A **fixed navigation bar** visible on all pages.
* Contains:

  * Logo: 🥐 *La Miètte*
  * Navigation links: Home, About, Menu, Gallery, Contact
  * CTA button: **“Visit Us”**
  * Hamburger menu (for mobile responsiveness)

👉 Features:

* Active link highlighting
* Responsive menu for smaller screens

---

## 🔷 3. Hero Section (Home)

```html
<section id="home" class="hero-section">
```

* The **first section users see**.
* Includes:

  * Background image (bakery/pastries)
  * Overlay for better text visibility
  * Heading: *“Baked with Love & Butter”*
  * Description text
  * CTA button: **Explore Our Menu**
  * Scroll indicator

👉 Purpose:

* Creates a strong **first impression** and brand identity.

---

## 🔷 4. About Section

```html
<section id="about">
```

* Displays:

  * Image of a baker kneading dough
  * Story of the bakery (heritage since 1987)

👉 Purpose:

* Builds **trust and emotional connection** with customers.

---

## 🔷 5. Menu Section

```html
<section id="menu">
```

* Shows bakery products in a **grid layout**.
* Each **menu card** includes:

  * Image
  * Item name
  * Description
  * Price

### Items listed:

* Butter Croissant
* Country Sourdough
* Celebration Cake
* Seasonal Fruit Tart
* Chocolate Éclair
* Specialty Coffee

👉 Purpose:

* Displays products in a **clean, attractive format**.

---

## 🔷 6. Gallery Section

```html
<section id="gallery">
```

* Image gallery with bakery visuals.
* Includes:

  * Different layouts (tall, wide, grid)
  * Hover overlay text (e.g., “Morning Display”)

👉 Purpose:

* Showcases **real bakery ambiance and products visually**.

---

## 🔷 7. Contact Section

```html
<section id="contact">
```

* Contains:

  * Bakery image
  * Address (Bandra, Mumbai)
  * Phone number
  * Email
  * Opening hours
  * CTA button: **Send Us a Message**

👉 Purpose:

* Helps users **visit or contact the bakery easily**.

---

## 🔷 8. Footer

```html
<footer>
```

* Displays copyright:

  * © 2025 La Miètte Artisan Bakery
* Branding message: *Made with butter and love*

---

## 🔷 9. External JavaScript

```html
<script src="script.js"></script>
```

* Adds interactivity like:

  * Navbar scroll effects
  * Hamburger toggle
  * Active link highlighting
